
## 逻辑1 是否允许乘客下单 及 计价

任何地方，默认情况，只允许独享，独享价为出租车打表计费标准。  
任何地方，默认情况，不允许拼车，不允许代驾。  
拼车指 需要1个座位 的拼车。 需要>=2两个座位的拼车 计价规则等同于独享（打表）  

```
如果( 起点是郑州四环内 且 终点在郑州四环内 ){
  不允许 独享
  不允许 拼车
  不允许 代驾
}

如果(起点 在 平顶山地级市范围 或 终点在 平顶山地级市范围){
  不允许 独享
  不允许 拼车
  不允许 代驾
  弹出叫车电话提示
}

如果( (起点在郑州四环内 或 起点在郑州周边特殊价格区域内) 且 终点在登封市区){
  允许 独享 ，价格分区 100 / 120 /160 / 180
  允许 拼车 ，价格分区  30 / 40 / 50 / 60
}  
```

## 逻辑2 计算拼车价格和独享价格  
伪代码：
```
if(终点在登封市区){
  if(起点 在 60元区域 ){
    拼车价格为60元 独享价格为180元
  }else if(起点 在 50元区域 ){
    拼车价格为50元 独享价格为160元
  }else if(起点 在 40元区域 ){
    拼车价格为40元 独享价格为120元
  }else if(起点 在 30元区域 ){
    拼车价格为30元 独享价格为100元
  }else if(起点 在 郑州四环内 且 起点 不在 郑州三环内 ){
    拼车价格为40元 独享价格为120元
  }else if(起点 在 郑州三环内 ){
    拼车价格为30元 独享价格为100元
  }
}
```
